EcoMetal - Sitio web con formulario conectado a Google Sheets

Este paquete mantiene el formato visual del HTML original y agrega el envio del formulario a Google Sheets.

Archivos:
- index.html
- assets/
- GoogleAppsScript/Code.gs
- Plantilla_Google_Sheet.csv
- PASOS_RAPIDOS.txt

Google Sheet:
https://docs.google.com/spreadsheets/d/1irq8I6rukonbBiKQ1n_rVrhoZ3gko5kWdIa5xyk2r10/edit?usp=sharing

Nota:
Para que el formulario funcione, debes publicar el Apps Script como Aplicacion web y pegar la URL /exec en index.html.
